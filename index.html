Discord Spellcast/Boggle Solver

This program was originally designed to solve the game of Spellcast on Discord, and later on I discovered the game is generally called 5x5 Boggle. I applied OOP principles when creating this program, making sure to cleanly organize the code into classes that mesh cleanly with eachother. I divide the 5x5 grid into nodes and connect them together. Then I recursively search for combinations of words that I will match to a dictionary of actual words I store in a hashset. The program runs and accepts user input through the terminal.

Asian American Studies Project

This project is my attempt at a text-based story game where the user follows the life of a vietnamese refugee and makes choices that impact the outcome of the game. The challenge for this project was properly organizing all of the text and dialogue so that I don't get lost. In order to implement choices, I had a function that handles user input and simply returns a binary choice. I decided to make each dialogue option its own function so that the functions can call eachother all the way down the line until they hit the end of the story, all without ugly loops.
